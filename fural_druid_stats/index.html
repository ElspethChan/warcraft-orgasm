    核心数据

    = 换算 ~ 约算

    主要属性

    耐力
- 1耐力 = 10血(人形)

敏捷
- 25敏 = 1暴击
- 1敏 = 1攻强(猫)
- 14.7敏 = 1闪避
- 1敏 = 2护甲

力量
- 1力量=2攻强

    防御属性

    护甲
- 35880满 (75%免伤)

防御等级(装备)
- 2.4防御等级 = 1防御技能

防御技能(面板)
- 带天赋默认350
- 无韧性415面板满免爆
- 60防御技能 = 1%免爆

闪避
- 19.92闪避等级 = 1%闪避几率

韧性
- 39.4韧性等级 = 1%免爆 = 2%暴击减伤
- 1韧性 ~ 1.5防御等级

免爆
- 默认面对73级BOSS 5.6%被暴击,减去天赋3%免爆,需要2.6%达到免爆
- 免爆需要额外 65防御技能 或者 104韧性
- 计算防御技能外的韧性补充: (415-面板防御技能)*1.6

    进攻属性

    命中等级(装备)
- 15.8命中等级 = 1%命中几率
- 1命中等级 ~ 2.7攻强

命中几率(面板)
- 9%满(73级敌人)

精准
- 硬上限221,软上限103
- 1精准 ~ 5.5攻强

暴击等级
- 1暴击等级 ~ 2.1攻强
- 22.1暴击等级 = 1%暴击几率

急速等级
- 15.77急速等级 = 1%急速
- 1急速 ~ 1.9攻强

    核心思路:

    TBC熊仇恨溢出,P1优先护甲和血量,次优先防御等级免爆和足够的命中;白字属性以敏捷和耐力为主;开荒偏重血甲,伐木期偏重堆敏捷命中和精准 配装根据属性配，P1暂时不需要太讲究BIS
